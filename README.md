# ASCII-ART Generator

Online [tool](http://ascii-web.herokuapp.com/) to create and download ASCII art from text using 3 different fonts.

Front-end:
  - HTML
  - CSS
  - JS

Back-end:
 - Golang
 - Docker

# Usage
To run server using Docker:
```sh
$ ./dockerize.sh
```
Or
```sh
$ go build
$ ./ascii-web
```
and open [http://localhost:8080](http://localhost:8080/) in your browser.
